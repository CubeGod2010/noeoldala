Halmazok, részhalmazok

Hogyan adhatjuk meg a halmazokat?:

  • Közös tulajdonság alapján, pl.
    A = {pozitív páros számok halmaza};
    B = {Európai Unió tagállamai};
    C = {Ady Endre versei}
  • Felsorolással, pl.
    D = {1; 2; 3; 4; 5};
    E = {1,2; -3,6; 5/7; 5π}; F =
    {Szent István, Péter, Aba Sámuel}

Végtelen elemszámű halmazokat is megadhatunk felsorolással, csak arra kell figyelni, hogy a folytatás irányát megfelelően jelöljük (...),
pl. G = {0; 1; 2; 3; ...};
H = {...; -2; -1; 0; 1; 2; ...} vagy H = {0; 1; -1; 2; -2; ...};
I = {10; 12; 14; 16; ...}/p>

A halmaz minden elemét csak egyszer soroljuk fel. Vegyük pl. az 1; 1; 1; 2; 2; 3 számsort. Ha ezt a számsort halmazban akarju kfeltüntetni, az így nézne ki: {1; 2; 3}. Viszont ha kellenek a konkrét értékek, pl. átlagszámítás céljából, akkor nem halmazról, hanem adatsokaságról, vagy a hat szám sorozatáról beszélünk.

Ha a szabályt valamilyen formulával, képlettel akarjuk megadni, azt is megtehetjük!
Tegyük fel, hogy olyan számokat akarunk megadni a J halmazba, melyek nagyobbak 7-nél, de viszont 15-nél kisebbek. Először is kell egy változó, mely a halmaz összes tagját helyettesíti, ezen esetben ez az x. Tehát eddig így néz ki a halmazunk:
J = {x}.
Az x után következő lépésként tennünk kell egy függőleges vonalat:
J = {x | }
A halmazban levő számokra a szabálynak így kell kinéznie:
7 < x < 15
Ezt a szabályt be kell illesztenünk a függőleges vonal után:
J = {x | 7 < x < 15}
Készen is vagyunk!

Nézzünk meg egy másik példát!
Tegyük fel, hogy egy előre megadott számsor számait akarjuk négyzetre emelni, és felsorolni őket egy halmazban. A számsorunk legyen az n számsor:
k = 1; 2; 3
Ebben a halmazleírásban most a tagokat x helyett n-nel helyettesítjük, mégpedig n²-nal, hisz nekünk a négyzetre emelt számokra van szükségünk. A halmaz betűje az L lesz. Eddigi információnk alapján így néz ki a halmaz:
L = {n² | }
most meg kell adjuk a szabályt:
L = {n² | n = 1; 2; 3;}
Tehát a szabály alapján az n helyére beírhatjuk a számsor egyes elemeit, és négyzetre emelhetjük őket!
